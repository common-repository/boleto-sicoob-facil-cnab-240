== Description ==

For Sicoob business customers with wallet service only.
Easily generate simple slips with registration through this plugin.
A page will be created. Check All Pages in your Wordpress.
or use the shortcode on the page you want to make the form available.
[gerar-boleto-sicoob-facil-cnab-importar]

The pages of this plugin are protected by the user password.
Must install plugin login-logout-shortcode-simple
Download: https://br.wordpress.org/plugins/login-logout-shortcode-simple/

Somente para clientes empresariais Sicoob com serviço de carteira de cobrança.
Gere facilmente boletos simples com registro através deste plugin.
Uma página será criada. Verifique Todas as Páginas no seu Wordpress.
ou use o shortcode na página que deseja disponibilizar o formulário.
[gerar-boleto-sicoob-facil-cnab-importar]

As páginas deste plugin são protegidas pela senha de usuário.
Obriagatório instalar o plugin login-logout-shortcode-simple
Download: https://br.wordpress.org/plugins/login-logout-shortcode-simple/

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for Boleto Sicoob Fácil Cnab).
* Alternatively, you can [download](https://downloads.wordpress.org/plugin/boleto-sicoob-facil-cnab.zip "Download Boleto Sicoob Fácil Cnab") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Boleto Sicoob Fácil Cnab` menu to configure the plugin.
